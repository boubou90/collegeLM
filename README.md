# Technologie au Collège

Site web pour les cours de technologie au collège.

## Structure du projet

```
my-vercel-project/
├── api/
│   └── index.ts
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   └── index.ts
│   ├── app.ts
│   └── styles/
├── vercel.json
├── package.json
├── tsconfig.json
└── README.md
```

## Développement

```bash
npm install
npm run dev
```

## Déploiement

Le site est automatiquement déployé sur Vercel à chaque push sur la branche main.
